# TDevEArchive
# 🗂️ TDevEArchive v26.1 Beta

**برنامج أرشيف وإدارة ملفات احترافي** | نسخة تجريبية

---

## 📌 عن البرنامج

TDevEArchive هو أداة متكاملة لإدارة وتنظيم الملفات والمجلدات، صممت لتكون بديلاً عملياً وحديثاً لمستكشف الملفات التقليدي، مع إضافات قوية مثل البحث الفوري، المعاينة المباشرة لأنواع متعددة من الملفات، وسلة محذوفات ذكية.

---

## ✨ الميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 📁 **الأقسام الرئيسية** | تنظيم الملفات في أقسام منفصلة وسهلة الوصول |
| 📥 **استيراد خارجي** | إضافة مجلد من أي مكان بالجهاز كقسم في الأرشيف |
| 📄 **إضافة ملفات ومجلدات** | إدراج ملفات ومجلدات داخل المجلد الحالي بسهولة |
| 🔍 **بحث سريع بفهرس ذكي** | نتائج فورية بدون الحاجة لمسح مجلد مجلد |
| 🖼️ **معاينة متعددة الأنواع** | صور، PDF، Word، Excel، فيديو، صوت |
| 🗑️ **سلة محذوفات متكاملة** | استعادة الملفات وحتى الأقسام الكاملة |
| ⭐ **المفضلة** | وصول سريع للمجلدات الأكثر استخداماً |
| 📊 **تقرير HTML** | شجرة كاملة لهيكل الأرشيف مع إحصائيات |
| ⌨️ **اختصارات لوحة المفاتيح** | تحكم كامل بدون ماوس |

---

## ⌨️ اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl + F` | بحث شامل |
| `Ctrl + D` | حذف (نقل للسلة) |
| `F2` | إعادة تسمية |
| `Backspace` | رجوع للخلف |
| `F5` | تحديث |

## 🖥️ متطلبات التشغيل

- **نظام التشغيل:** Windows 7 / 8 / 10 / 11
- **مساحة التخزين:** ~50 MB
- **لا يحتاج تثبيت** – يعمل مباشرة بعد فك الضغط

---

## 📥 طريقة التشغيل

1. قم بتحميل الملف المضغوط من الرابط المرفق
2. فك الضغط في أي مجلد على جهازك
3. شغّل الملف `TDevEArchiveV261Beta.exe`
4. البرنامج سينشئ تلقائياً مجلدات `Archive_Storage` و `Archive_Trash`

> ⚠️ **ملاحظة:** قد يظهر تحذير من Windows Defender – هذا طبيعي للبرامج غير الموقعة. اضغط على "تشغيل على أي حال" أو أضف البرنامج إلى الاستثناءات.
